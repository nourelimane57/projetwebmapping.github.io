Agriculture au MAROC

réalisation d'une Connection directe à la base de données Postgres avec une extension PostGIS

Pour réaliser les requêtes spatiales merci de mettre la géometrie sous forme de GeoJSON sou le nom geom comme : ST_AsGeoJSON(geom) AS geom

exemple de requêtes spatiales à essayer

SELECT region, ST_AsGeoJSON(geom) AS geom FROM regions_2015

SELECT region, ST_AsGeoJSON(geom) AS geom FROM regions_2015 WHERE prd_bledur > 1000
